Purpose & Aim

This mod adds an average rating line to the product info page. Average rating is taken from reviews for current product ID and current language and is displayed as number rounded to 2 decimal digits and the "stars" image just after Current Reviews line. New "half-star" images was added to display average ratings with 1, 1.5, 2, 2.5, 3, 3.5, 4, 4.5 and 5 stars. See real screenshots in the /screenshots/ folder of this contribution.

Installation Instructions
  1. BACK UP YOUR STORE FILES AND DATABASE BEFORE GETTING STARTED!!
  2. Rename YOUR_ADMIN to be the name of your admin
  3. Upload the files keeping the directory structure
  4. Load Your Admin, the module will self install
  5. Modify your product_info/document_info/etc pages with teh following change
    • Find the statment that looks like this: if($flag_show_product_info_reviews_count == 1){ ...BUNCH OF STUFF... }
    • Replace with: if($flag_show_product_info_reviews_count == 1){ echo average_product_reviews_string((int)$_GET['products_id']); }


Uninstall Instructions
  1. BACK UP YOUR STORE FILES & DATABASE BEFORE UNINSTALLING!!
  2. Revert your file Changes
  3. Remove the installation files from your server
  4. Run the uninstall.sql in the admin Tools->SQL Patches
