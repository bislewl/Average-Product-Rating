Purpose & Aim

This mod adds an average rating line to the product info page. Average rating is taken from reviews for current product ID and current language and is displayed as number rounded to 2 decimal digits and the "stars" image just after Current Reviews line. New "half-star" images was added to display average ratings with 1, 1.5, 2, 2.5, 3, 3.5, 4, 4.5 and 5 stars. See real screenshots in the /screenshots/ folder of this contribution.

Installation Instructions
  1. BACK UP YOUR STORE FILES BEFORE GETTING STARTED!!
  2. Extract the contents of the zip file to your PC, leaving the folder/file structure intact
  3. Rename all of the YOUR_CUSTOM_TEMPLATE folders to match the name of your template folder
    • Either /includes/languages/english/YOUR_CUSTOM_TEMPLATE or /includes/languages/czech/YOUR_CUSTOM_TEMPLATE depending on your language preference
    • includes/templates/YOUR_CUSTOM_TEMPLATE
  4. If any changes to the above files have been made by you or another addon, you must merge those changes into these files prior to uploading them to your server.
    • A file merging program like WinMerge or BeyondCompare is very helpful in this process. Both are free and available online.
    • If you prefer to to merge the Average Product Rating v2.0 file changes into your existing files instead, you'll find that all these changes are clearly commented in the code.
      • Look for code beginning with // 2P added BOF - Average Product Rating and ending with // 2P added EOF - Average Product Rating
  5. Once all your file merging is completed (if any), copy the entire contents of the /includes folder to your store's includes folder.
    • If you have no need for both language files, you may delete the one you won't be using prior to uploading the other contents of your /includes folder
  6. Optional Step: For better vertical alignment of the "stars" images you can add the following code to the end of your stylesheet
    • Your template's custom stylesheet may be found on your server at /includes/templates/YOUR_CUSTOM_TEMPLATE_FOLDER/css/stylesheet.css
    • /* Average Product Rating */
      .reviewCount img {
      vertical-align: middle;
      }


Uninstall Instructions
  1. BACK UP YOUR STORE FILES BEFORE UNINSTALLING!!
  2. Either remove the installation files from your server or remove the Average Product Rating v2.0 code from the installation files by carefully following steps 3 and 4 below
  3. If you have made no changes to any of the installation files since you installed Average Product Rating v2.0, then you may safely remove the installation files from your server, HOWEVER
    • The below files are CORE ZEN CART FILES and MUST BE REPLACED with their counterparts from the Zen Cart v1.5.0 fileset after you delete the Average Product Rating v2.0 versions from your server:
      • /includes/modules/pages/document_general_info/main_template_vars.php
      • /includes/modules/pages/document_product_info/main_template_vars.php
      • /includes/modules/pages/product_free_shipping_info/main_template_vars.php
      • /includes/modules/pages/product_info/main_template_vars.php
      • /includes/modules/pages/product_music_info/main_template_vars.php
  4. If you have made changes to any of the installation files since you installed Average Product Rating v2.0, then it it best to remove the code from the files, rather than remove the files themselves.
    • All Average Product Rating v2.0 file changes are clearly marked in the code, making it easy to find and remove. Look for code beginning with // 2P added BOF - Average Product Rating and ending with // 2P added EOF - Average Product Rating
  5. If during installation, you proceded with Optional Step 6, you may remove that code from your stylesheet
